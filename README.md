SublimeTextSnippets
===================

Auto completion snippets for Sublime Text 2


### WORDPRESS

SNIPPET  | PRODUCES
------------- | -------------
wp_child_dir  | get_stylesheet_directory_uri()
wpchild  | style.css for child theme
wp:page | add_menu_page()
ecss | embed CSS
ejs | embed JS
update_option | update_option() for siteurl and home
